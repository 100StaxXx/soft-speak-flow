# Lovable AI Model Information

## Yes, Lovable Uses Claude Sonnet 4.5

This project is built on the **Lovable.dev** platform, which uses **Claude Sonnet 4.5** by Anthropic as its AI engine.

> **Note**: The question "Are you using Claude Sommet 4.5?" contains a typo. The correct name is "Claude **Sonnet**" (not "Sommet").

## Current Model: Claude Sonnet 4.5

As of late 2025, Lovable.dev is powered by **Claude Sonnet 4.5**, Anthropic's most advanced coding AI model to date.

### Key Capabilities

- **Industry-leading coding performance**: 77.2% on SWE-bench Verified, up to 82% with parallel compute
- **Multi-step reasoning**: Superior performance on complex, multi-day coding tasks
- **Long-context understanding**: Maintains focus and accuracy across extended workflows
- **Enhanced instruction following**: Better at understanding and implementing user requirements
- **Computer interface usage**: Improved ability to interact with development tools and environments
- **Reduced vulnerabilities**: Lower rates of prompt injection and improved safety

## Model Evolution Timeline

### 2024: Claude 3.7 Sonnet
- Lovable's initial AI engine
- Strong reliability and design capabilities
- Solid foundation for code generation

### May 2025: Claude Sonnet 4
- Major upgrade with improved multi-step reasoning
- Enhanced coding accuracy
- Better instruction following

### September 2025: Claude Sonnet 4.5 (Current)
- Best coding AI model available
- Generational leap in performance
- Powers Lovable's latest capabilities including:
  - Native code execution
  - Build checkpoints
  - Improved memory management
  - AI image generation for design tools

## Impact on This Project

Because this project is built with Lovable, all AI-assisted development features leverage Claude Sonnet 4.5's capabilities:

- **Code Generation**: High-quality, production-ready code
- **Natural Language Understanding**: Accurate interpretation of requirements
- **Design Integration**: AI-powered design themes and real-time editing
- **Long-term Context**: Maintains consistency across complex, multi-file changes
