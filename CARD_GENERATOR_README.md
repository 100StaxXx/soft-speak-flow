# Card Test Generator

Quick tool to visualize card stats and formulas.

## Usage

```bash
npx tsx card-test-generator.ts
```

Generates 5 sample cards per stage (0, 5, 10, 15, 20) with:
- Mind/Body/Soul stats
- Energy costs
- Bond levels
- Rarity tiers

Tests with 3 attribute profiles (Beginner/Average/Advanced) to show stat ranges.
